multi_session
==============

Creates multiple sessions for rails.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'multi_session'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install multi_session
```
